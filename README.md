Voice Assistant using Python
Description :
This project is a simple Voice Assistant built with Python. It listens to the user’s voice commands and responds with spoken output. The assistant can perform basic tasks like telling the time and opening websites.

Technologies Used :
Python,
   SpeechRecognition,
      pyttsx3

Features :
Voice command recognition,
  Text-to-speech response,
    Tels current time,
       Opens websites like Google or YouTube

How to Run :
Install required libraries:
pip install SpeechRecognition pyttsx3 pyaudio

Conclusion :
This project demonstrates a basic voice-controlled assistant using Python.
